# DeepLearn Pro

DeepLearn Pro is an AI-powered learning platform designed to master Machine Learning and Deep Learning. It features a comprehensive curriculum inspired by the world-class courses at `deeplearning.ai`.

## 🚀 Key Features

- **Standardized Curriculum:** Detailed paths for Supervised Learning, Advanced Algorithms, and Deep Neural Networks.
- **AI Learning Mentor:** Get real-time AI feedback and explanations on complex topics.
- **Interactive Playground:** Visualize neural network architectures and experiment with parameters.
- **Hands-on Projects:** Build models from scratch, including Linear Regression and Deep Neural Networks.
- **Adaptive Quizzes:** Test your knowledge with dynamically generated questions.
- **AI Project Generator:** Generate personalized project ideas to build your portfolio.

## 📚 Integrated Curriculum (DeepLearning.AI)

The platform now features detailed content for:
- **Supervised Machine Learning:** Regression and Classification.
- **Advanced Learning Algorithms:** Neural Networks, Decision Trees, and Ensembles.
- **Unsupervised Learning:** Clustering, Anomaly Detection, and Recommenders.
- **Deep Learning Foundations:** Vectorization, Backpropagation, and Optimization.

## 🛠️ Technology Stack

- **Framework:** Next.js 15+ (App Router, Turbopack)
- **Styling:** Tailwind CSS / Shadcn UI
- **Backend:** Firebase (Firestore, Auth)
- **AI Engine:** Google Gemini AI (via Genkit)
- **Language:** TypeScript

## 🏁 Getting Started

### Prerequisites
- Node.js 18+
- Firebase Project

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/lucifer55555555/DeepLearnPro.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   Create a `.env` file with:
   ```env
   GEMINI_API_KEY=your_key_here
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
